# PAN token
ERC20 tokent with upgradeability and MultiSig governance


## Requirements
- node.js v9 (up)
- npm

## Setup
```sh
npm i
```

## Testing
```sh
npm run test
```
Run tests with [coverage](./coverage/index.html):
```sh
npm run test:coverage
```
